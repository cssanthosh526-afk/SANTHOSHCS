# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in **VTU-Internship-Dairy**, please report it **privately** so it can be addressed before it becomes public.

### How to Report

1. Email the project maintainers at: `pradhyumnar26@gmail.com`
2. Include the following information:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested fixes (optional)
3. Please do **not** create a public GitHub issue for security vulnerabilities.

We will respond within 72 hours.

---

## Supported Versions

We currently support all versions in the `main` branch.

---

## Security Best Practices

- Always keep your installation updated to the latest version
- Follow safe authentication and network practices
- Report suspicious behavior or bugs via the private security contact
